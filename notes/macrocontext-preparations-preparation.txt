# 2. Attribute Clusters (Universal Entity Template)

These are the **core attribute vectors** every preparation-form entity should cover.

---

# A. Identity Attributes (What It Is)

Defines the entity.

Subtopics:

* Definition of the preparation form
* Difference from other herbal forms
* Role in herbal medicine
* Classification within herbal preparations

Important vectors:

```
dosage form
herbal preparation type
delivery format
```

Example triples:

```
Herbal Tincture → Category → Liquid herbal extract
Herbal Capsule → Category → Solid dosage form
```

---

# B. Functional Attributes (Why It Exists)

Describes **the function of the form**.

Subtopics:

* Delivery of plant compounds
* Preservation of herbal material
* Bioavailability
* Ease of administration

# 4. Microsemantic Tip (Important)

Every page should start with a **clear entity definition**.

Example for tinctures:

> A herbal tincture is a concentrated liquid extract of medicinal plants prepared by soaking herbs in alcohol or another solvent to dissolve their active compounds. Tinctures are widely used in herbal medicine because they preserve plant constituents and allow precise dosing.

This establishes:

* entity identity
* category
* attributes

which helps **entity recognition and retrieval**.
