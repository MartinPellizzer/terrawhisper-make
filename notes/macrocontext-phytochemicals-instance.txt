
Below is a **reusable Koray-style template** for **any phytochemical class page** such as:

```
/phytochemicals/alkaloids.html
/phytochemicals/flavonoids.html
/phytochemicals/terpenoids.html
/phytochemicals/tannins.html
/phytochemicals/saponins.html
```

The goal is to make **each phytochemical class stand alone as an entity node** while still being connected to the larger SCN:

```
Herbal Medicine
→ Phytochemicals
→ Classification
→ Specific Phytochemical Class
```

The page should therefore cover **all primary attribute vectors of a phytochemical entity type**.

# 4. Microsemantic Tip (Critical)

Your **first paragraph should define the compound class immediately**.

Example for **alkaloids**:

> Alkaloids are nitrogen-containing phytochemicals produced by plants as secondary metabolites, many of which have potent biological activity in humans. In herbal medicine, alkaloids are responsible for numerous therapeutic effects such as pain relief, antimicrobial activity, and nervous system modulation, and they occur in medicinal plants such as poppy, belladonna, and goldenseal.

This establishes:

* entity identity
* chemical class
* herbal medicine relevance
